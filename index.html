Install

Hydrogen is a modern CSS foundation. We avoid the use of the word "framework", because it implies bias. Hydrogen is entriely unbiased and offering up only a starting point with the intention of being overriden. It is small enough to be justifiable on small projects, and customizable enough to use on large applications.


To get started simply download the base CSS file (~2.4KB / 943 bytes gzipped) and the optional grid units file.

Grid

By default, Hydrogen's grid provides you with the foundation to build fully custom, responsive grids. Enabling you to truly program for the responsive medium that is the web.

<style> @media screen and (min-width: 50em){ .content { width: 75%; } .sidebar { width: 25%; } } </style> <div class="clr-g"> <div class="clr-u content"> ... </div> <div class="clr-u sidebar"> ... </div> </div>

Grid Units
